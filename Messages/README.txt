Messages.zip
------------
Contains six prefabricated message#.xml files (1 through 6) to use for completing all eight tutorials in the Red Hat JBoss Fuse: Tooling Tutorials guide. If you are working through tutorial 1 “To Create a New Route,” you need not install message1.xml.

Unzip Messages.zip and install the prefabricated message#.xml files into your CBRroute project’s src/data/ folder.
